Proximity

Utah.gov/Visit

In this example you can see that the web-designers have done a good job using the rules of proximity to help the user easily find what interests him or her. This is done most nobably with grouping like items together. This creates less clutter while still giving the user ideas of what to explore next. The user will intuitively understand that items are related to each other simply because they are closer to each other.
